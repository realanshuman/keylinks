KeyLinks

KeyLinks is an open-source tool for generating secure, shareable redemption links for activation keys, gift cards, software licenses, coupons, and other digital products.

Instead of sharing the actual code, create a unique link that securely reveals it when opened.

Features

* Secure redemption links
* Single and bulk generation
* Password protection
* Expiration and usage limits
* QR code generation
* Admin dashboard
* Analytics and activity logs

Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* NestJS
* PostgreSQL
* Prisma
* Redis
